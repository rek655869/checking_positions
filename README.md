# Проверка должностей (CatWar)

Пользовательский скрипт для автоматизации проверки должностей игроков. Позволяет быстро собрать информацию о персонажах по их ID, сгруппировать по должностям и скопировать списки имен и/или ID. Добавляет кнопку **«Проверка должностей»** на страницу редактирования блога и в линковку.

### Установка
**[Установить скрипт (прямая ссылка)](https://raw.githubusercontent.com/rek655869/checking_positions/refs/heads/master/script.user.js)**

> **⚠️ Если установка не начинается или загрузка длится слишком долго:**
> 
> Прямая ссылка заставляет менеджер расширений загрузить скрипт, но это может не сработать из-за различных ограничений (например, Tampermonkey может не загружать без VPN). В таком случае можно установить скрипт самостоятельно.
> 1. Откройте менеджер расширений (Tampermonkey, Violentmonkey или Greasemonkey).
> 2. Нажмите кнопку **«Создать новый скрипт»** (обычно иконка с плюсом).
> 3. Скопируйте весь код скрипта **[этой ссылке](https://github.com/rek655869/checking_positions/blob/master/script.user.js)** (Ctrl+A -\> Ctrl+C или кнопка "Copy raw file").
> 4. Удалите стандартный шаблон в редакторе, вставьте скопированный код и нажмите **«Сохранить»** (Ctrl+S).

## Как пользоваться

### 1\. В линковке

  * Вставьте список ID в поле. Допускается вставлять и текст, при проверке он будет очищен и никак не повлияет на работу скрипта.
  * Нажмите на кнопку и выберите, нужно ли группировать по должности. При включённой группировке игроки с одной должностью будут выведны в одном столбце таблицы. При выключенной - одним списком в том порядке, в каком были вставлены ID.
    + Введите максимальное количество столбцов. По умолчанию - 3. Подберите то количество, которое будет удобно с вашим размером экрана и масшабом.
  * Нажмите **«Проверить»** и дождитесь заполнения шкалы прогресса.

  ### 2\. В блоге

  * Выберите тип ссылок, которые нужно учитывать: [link...] и/или [url=cat...]. Скрипт соберет все ID, которые встречаются в указанном типе ссылок в блоге.
  * Выберите, нужно ли группировать по должности, как описано в предыдущем пункте.
  * Нажмите **«Проверить»** и дождитесь заполнения шкалы прогресса.

### 3\. Работа с результатами

  * Используйте чекбоксы в шапках таблиц, чтобы выделить всех игроков конкретной должности.
  * Кнопка **«Сбросить выделение»** очистит все выбранные пункты.
  * Кнопки копирования сохраняют выделенных игроков в буфер обмена.
